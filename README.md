## 📋 Proje Hakkında

Rezervasyon Sistemi, yakınındaki kafe ve restoranları listeleyen ve kullanıcıların bu mekanlara kolayca masa rezervasyonu yapabilmesini sağlayan demo bir web uygulamasıdır.

---

## 🚀 Özellikler

- 🗂 8 farklı demo kafe & restoran listesi
- 🔍 Tür bazlı filtreleme (Kafe / Restoran / Fast Food)
- 📅 Tarih, saat ve kişi sayısı seçimi
- ✅ Rezervasyon onay ekranı
- 💾 Rezervasyonlar localStorage'a kaydedilir
- 🗑 Rezervasyon iptal etme
- 📱 Tam responsive tasarım (mobil uyumlu)
- 🌿 Modern Fresh renk teması
